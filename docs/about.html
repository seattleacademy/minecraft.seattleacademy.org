Connecting to Our Minecraft Server

Copy the Saas server address: minecraft.seattleacademy.org.

Step 1: Open Minecraft and click on Multiplayer

Step 2: Click Add Servers

Step 3: Enter Server Information

Enter the name of the Saas server which is also minecraft.seattleacademy.org in the Server Name box.

In the Server Address box, paste the Saas server address.

Click Done.

Step 4: Connect to the Server

Click on the server in the list and select Join Server to connect!
