Seattle Academy Campus Minecraft Server

This project was developed in the Spring of 2020 by Gary Anderson's Software Development class.

Unable to come to the campus because of the Corona-Virus lockdown, the 25 Spring Software Development students created their own highly detailed virtual campus as a final project. This brought togehter all of our prior technology projects and created an impressive collaboative work that was fun and usuable for all in the future.
